# Django CMS

A simple Django-based Content Management System (CMS) project.

## 🔧 Tools Used

- Python 3.12
- Django
- [Poetry](https://python-poetry.org/) for dependency management

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Miladkhoshdel/django-cms
cd django-cms
```

```bash
python manage.py loaddata initial_settings
```

## 📦 Dependency Management with Poetry

This project uses **Poetry** to manage dependencies and isolate virtual environments.
